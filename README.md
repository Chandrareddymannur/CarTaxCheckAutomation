## CarTaxCheck Automation

### Technology Stack

- Java 8
- Cucumber
- Selenium Webdriver
- Pico Container
- Maven
- OpenCSV
- AssertJ
- Slf4j


### IDE
 
- IntelliJ IDEA


### Browser Support

- Google Chrome


### Run Tests

- `mvn clean test`


### Reports
- Generated reports are under target/cucumber-reports/cucumber.html/index.html


### Screen Recordings
- Test execution is recorded and can be found under screen_recordings folder.
